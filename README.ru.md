## Что это

`index.html` — **одностраничный** тренажёр слепой печати. Откройте файл в современном браузере и сразу тренируйтесь: сервер и дополнительные ресурсы не нужны.

За один заход показывается фраза из **10–15 слов** (через пробел) на одном языке. По желанию можно включить **цифры и знаки** — в фразу добавятся 1–3 фрагмента только из символов. Набор — посимвольно. Словарь **согласован между языками** (один смысл — строки RU, EN, ES, IT), поэтому можно тренироваться на одном языке и подсматривать переводы на другие. Экранная клавиатура соответствует раскладке (**QWERTY** для EN/ES/IT, **ЙЦУКЕН** для русского) и подсвечивает нажатые клавиши.

## Возможности

### Языки

| | |
| --- | --- |
| **Интерфейс** | Весь UI (подписи, кнопки, сообщения, заголовки таблиц): **Русский**, **English**, **Español**, **Italiano**. |
| **Язык слов** | В каждом раунде слова только на одном из четырёх языков. |
| **Связка** | При смене языка интерфейса язык слов подстраивается под него; при необходимости его можно сменить отдельно. |
| **Память** | Язык интерфейса хранится в `localStorage` (`bk_ui_lang`). При первом заходе по возможности учитывается язык браузера. |

### Раскладки

| Группа языков | Раскладка | Метки домашнего ряда |
| --- | --- | --- |
| Русский | ЙЦУКЕН (JCUKEN) | **А** и **О** |
| Английский, испанский, итальянский | QWERTY (ES/IT — одна карта клавиш) | **F** и **J** |

### Подсказки при наборе

| Действие | Результат |
| --- | --- |
| **Десктоп** | Наведение на слово → всплывающая подсказка с переводами на *остальные* три языка. |
| **Сенсор** | Долгое нажатие (~0,5 с) на слово → та же подсказка. |

Фразы собираются из общего словаря: одна семантическая строка — четыре перевода.

### Статистика

| Блок | Содержимое |
| --- | --- |
| **Сессия** (под зоной набора) | Число слов, символы, верные / неверные нажатия, время. |
| **Сводка** (четыре карточки) | По языкам: попытки, суммарные верные/ошибочные нажатия, идеальные и с ошибками фразы. |
| **История** | Последние **100** завершённых фраз: фрагмент текста, язык, время, ошибки, результат. |

Все данные — в `localStorage` (`bk_typing_stats_v1`).

### Набор текста

- **Полоса фразы** — *уже набрано* (приглушено) · *текущий символ* (акцент) · *впереди* (легче).
- **Фокус** — если фокус ушёл, **первая печатная клавиша** возвращает фокус в зону набора и учитывается как ввод.
- **Вставка** — вставка без фокуса переносит фокус в зону практики (содержимое буфера как текст упражнения не подставляется).
- **Новая фраза** — случайная фраза на текущем языке слов.
- **Ошибки** — короткая красная вспышка; клавиши на схеме реагируют на **keydown / keyup** глобально.

## Технические детали

| Тема | Подробности |
| --- | --- |
| **Сборка** | Весь HTML, CSS, JavaScript и словарь — в **`index.html`**. |

| **Хранилище** | `bk_ui_lang`, `bk_typing_stats_v1`; история не более 100 записей. |

## Краткая справка

| | |
| --- | --- |
| **Языки** | RU / EN / ES / IT — интерфейс и слова |
| **Раскладки** | ЙЦУКЕН (RU), QWERTY (EN·ES·IT) |
| **Длина фразы** | 10–15 слов (опционально 1–3 блока только из знаков) |
| **Переводы** | Наведение (мышь) или долгое нажатие (тач) на слово |
| **Запуск** | Откройте **`index.html`** в этой папке (или отдайте каталог **`blind-keyboard/`** как корень сайта) в браузере |
